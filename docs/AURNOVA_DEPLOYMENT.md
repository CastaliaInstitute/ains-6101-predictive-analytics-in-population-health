# Aurnova Course Deployment Model

This repository delivers **AINS6101 / AIN6101 — Predictive Analytics in Population Health** as a canonical Aurnova MSAI course package.

It follows the AINS6003 deployment pattern:

- canonical Castalia course source repository;
- GitHub Pages course site at `https://ains6101.courses.castalia.institute/`;
- optional Aurnova instructor repository fulfillment;
- cohort and student assignment repository workflows;
- Populi roster synchronization hooks.

Canonical Aurnova instructor repository name:

```text
Aurnova/ain6101-instructor
```

Cohort repository examples:

```text
Aurnova/ain6101-fall2026-course
Aurnova/ain6101-fall2026-m01-template
Aurnova/ain6101-fall2026-m01-<github_username>
```

Before production, confirm Aurnova GitHub Enterprise settings, Populi API credentials, private Pages requirements, and grade return workflow.
